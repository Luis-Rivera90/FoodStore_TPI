# FoodStore TPI - Programación 2

Sistema de gestión de pedidos para tienda de alimentos desarrollado en Java con OOP.

## Tecnologías
- Java 21
- NetBeans IDE

## Estructura del proyecto
- `foodstore.entities` — clases del modelo (Base, Categoria, Producto, Usuario, Pedido, DetallePedido)
- `foodstore.enums` — enumeraciones (Rol, Estado, FormaPago)
- `foodstore.interfaces` — interfaz Calculable
- `foodstore.exception` — excepciones propias
- `foodstore.services` — lógica de negocio
- `foodstore.menu` — menú de consola

## Cómo ejecutar
1. Clonar el repositorio
2. Abrir el proyecto en NetBeans
3. Ejecutar `Main.java`
